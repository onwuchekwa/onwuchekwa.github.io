Navigation

Navigation

Navigation will sit near the top of the page, between the header and the main elements. It will have the following characteristics:

  • Unselected item: It will be Golden Tainoi (#ffc857) background with Valhalla (#1f2041) for the text.
  • Hovered Item: Reversed color scheme, with Valhalla (#1f2041) background with Golden Tainoi (#ffc857) for the text.
  • Selected Item: It will be Golden Tainoi (#ffc857) background with dark red (#a50404) for the text.
